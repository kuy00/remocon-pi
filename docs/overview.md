# 프로젝트 개요 — remocon-pi

라즈베리파이 + `pigpio` 기반으로 **에어컨 IR 리모컨 신호를 녹화 → 분석 → 재생**하여 에어컨을 제어하는 프로젝트.

## 동작 원리

```
[리모컨] ──(GPIO13 수신)──▶ 녹화 ──▶ 분석 ──▶ 재생 ──(GPIO18 송신)──▶ [에어컨]
```

핵심 철학: **프로토콜을 완전히 해독하지 않아도, 녹화한 원시 펄스를 그대로 되쏘면(replay) 에어컨이 동작한다.** 따라서 체크섬 등 미해독 바이트와 무관하게 제어가 가능하다. 분석/모니터링 단계는 검증·이해를 위한 보조 도구이며, 실제 제어는 송신 스크립트만으로 충분하다.

## 목표: 수집 → 학습 → 예측 생성

충분히 수집한 데이터로 **규칙을 자동 학습**하고, 수집하지 않은 조합도 **합성(synthesis)** 해
송신하는 것이 목표다. 특정 에어컨에 의존하는 값(주소·바이트 배치·체크섬)을 **하드코딩하지 않고**,
어떤 리모컨이든 수집 데이터만으로 규칙을 도출한다.

```
ir_collect → ir_learn → (ir_synth) → ir_send
  수집         학습        합성         송신
```

| 단계 | 스크립트 | 역할 |
|------|----------|------|
| 1. 수집 | `ir_collect.py` | `sweep.json` 축을 스윕, 설정당 8회 반복 + 신뢰도 게이트 → `dataset/` |
| 2. 학습 | `ir_learn.py` | `dataset/` 자동 분석: 필드 발견(선형/룩업)·체크섬 탐색 → `model.json` |
| 3. 합성 | `ir_synth.py` | 가장 가까운 수집본을 템플릿으로 미수집 온도의 바이트→펄스 생성, 자가검증 |
| 4. 송신 | `ir_send.py` | 저장 펄스 재생(replay) + 합성 송신 |
| 보조 | `ir_monitor.py` | 실시간 디코딩/대조 검증 |
| 공통 | `ir_codec.py` | raw 펄스 ↔ 비트/바이트, 다수결·신뢰도 (순수 로직) |
| 공통 | `ir_io.py` | pigpio 저수준 I/O — 수신 collector + 송신 캐리어/파형 |

### 핵심 설계
- **헤더 타이밍 하드코딩 제거** — 긴 무신호 갭으로만 프레임 구분
- **고반복 + 신뢰도** — 설정당 8회 수집, 반복 일치율 낮으면 자동 재촬영 (체크섬 해독엔 깨끗한 데이터가 전제)
- **차분 분석** — 한 파라미터만 바꿔 수집 → 어느 비트가 어느 파라미터인지 자동 격리

### 현재 상태
- ✅ 수집·학습·공통 코덱 구현 완료. F1류 필드(주소/모드/전원/온도)는 자동 해독, 온도는 선형 예측 가능
- ✅ F2 B4·B5 = **프레임 전체 합이 (mode,power)그룹 상수인 2바이트 체크섬**으로 해독.
  `ir_learn.py`가 `frame_sum_pair`로 **자동 분류**(complex 0 달성). 온도 +dT → B4 -dT 로 합 보존
- ✅ `ir_synth.py` 합성기 구현·**실송신 검증 완료** — 미수집 온도를 가장 가까운 수집본 템플릿으로
  합성해 송신, 에어컨이 정상 동작(전체합 체크섬을 검증하므로 B4/B5 분할이 실측과 달라도 통과)
- ℹ️ B5의 미세분할 규칙 자체는 미해독이나 전체합이 보존되어 합성에는 영향 없음

## 데이터

| 산출물 | 내용 |
|------|------|
| `dataset/` | `ir_collect.py` 출력 — 설정별 `params` + 반복 raw 펄스 + 신뢰도 |
| `model.json` | `ir_learn.py` 출력 — 학습된 프로토콜 규칙 |

## 하드웨어 요약

- 수신: KY-022 IR 수신기 OUT → GPIO **13** (BCM)
- 송신: IR LED(트랜지스터 구동) → GPIO **18** (BCM), 38kHz 캐리어
- `pigpiod` 데몬 실행 필요

자세한 배선은 [hardware.md](hardware.md), 사용법은 [usage.md](usage.md), 데이터/프로토콜 구조는 [data-format.md](data-format.md) 참고.
